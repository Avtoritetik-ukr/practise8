# practise8
#  JWT Authentication & RBAC Workshop

Цей репозиторій містить реалізацію **REST API** з автентифікацією через **JWT (JSON Web Tokens)** та авторизацією на основі ролей (**RBAC**). Проєкт створено на базі **ASP.NET Core Minimal API**.

## Технології
* **Мова:** C# (.NET 10)
* **Фреймворк:** ASP.NET Core Minimal API
* **Бібліотека:** Microsoft.AspNetCore.Authentication.JwtBearer

## Як запустити проєкт

1. **Клонувати репозиторій:**
git clone https://github.com/Avtoritetik-ukr/practise8.git
2. **Перейти в папку з кодом:**
зайти в папку куди було скопійовано репозиторій, відкрити термінал в папці practise8 та вписати cd src/src
3. **Запустити сервер**
після кроку 2 вписати в той же термінал dotnet run
