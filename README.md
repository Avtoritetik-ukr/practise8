#  JWT Authentication & RBAC Workshop

Цей репозиторій містить реалізацію **REST API** з автентифікацією через **JWT (JSON Web Tokens)** та авторизацією на основі ролей (**RBAC**). Проєкт створено на базі **ASP.NET Core Minimal API**.

## Технології
* **Мова:** C# (.NET 10)
* **Фреймворк:** ASP.NET Core Minimal API
* **Бібліотека:** Microsoft.AspNetCore.Authentication.JwtBearer
